# A Simple castbar addon for WoW Midnight

Install this the same as any other WoW Addon. 

`/jcb` opens the configuration screen:

<img width="545" height="734" alt="image" src="https://github.com/user-attachments/assets/9a78234a-82df-4b67-b119-985d463bee7b" />

The cast bars are always centered, so you only can move them up and down. You can configure:
* Font
* Texture
* Width
* Vertical position on the screen
